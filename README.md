Retail Sales Analytics using Python

Developed a retail analytics system using Python, Pandas, and Matplotlib to process and analyze sales data from a CSV dataset containing 500+ records.

Performed data cleaning and preprocessing, including handling missing values, standardizing text fields, and converting date and numeric formats.

Implemented feature engineering by calculating revenue metrics from sales transactions.

Conducted exploratory data analysis (EDA) to identify top-selling products, highest-revenue cities, customer spending patterns, and category-wise performance.

Created visualizations such as bar charts, line charts, and pie charts to represent sales trends and business insights.

Generated automated reports summarizing total revenue, top products, top customers, and regional sales performance.

Applied Object-Oriented Programming (OOP) concepts to design a modular and reusable analytics pipeline.


Technologies Used: Python, Pandas, Matplotlib, NumPy, CSV Handling, OOP
